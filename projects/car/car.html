NeuroCar

Why?

Modern computing is very taxing and inefficient in traditionally decision making, so what can we do about it? Spiking neural networks require low power and operate very efficiently since it uses models of real neurons to compute. This method of computation is neuromorphic since it is modeled after the brain.

How?

So this is all cool theory but how was it done? First, a car with a camera module was made from the Sunfounder PiCar. The reason this car was used was that it natively supported the Raspberry Pi and that is what we were going to use to run the software and to process and interpret the data from the camera module. This was done with RatSLAM which is a form of Simultaneous Localization and Mapping (SLAM) inspired by rat brains. This is a neuromorphic method of mapping and tracking location within an environment.

What?

With all of this planning, the chassis was made and tested by first con trolling the car with a keyboard and then with an Xbox controller for more precise inputs since it's analog. Next steps for the upcoming semester include getting OpenRatSLAM, the open-source version of RatSLAM, running on the Raspberry Pi, and then getting this to function on an FPGA for even more efficiency. One final goal for down the line is to get this to function on an FPAA which consists of analog block arrays which resemble human neural networks. With this, brain-mimicking software can be run on brain-mimicking harware for the most seamless and effecient experience.
